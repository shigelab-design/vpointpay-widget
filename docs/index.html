このウィジェットでできること

ホーム画面で即・残高チェック

VポイントPayアプリを開かなくても、ホーム画面のウィジェットを見るだけで残高がわかります。

オシャレなのにシンプルで見やすい残高表示

デザイン性は持ちつつ余計な情報を入れず、ホーム画面上で邪魔にならない1×2・2×2サイズのレイアウトです。太字・影付きの残高表示で、パッと見てすぐに金額が認識できるようデザインしています。

Gmailの通知を読み取るだけなのでセキュリティ◎

Vポイントpayを利用/チャージの時に届くGmailの通知を読み取るだけなので、外部と一切接続なし。セキュリティは最高級です。 ※Vポイントpayアプリの通知をGmailに届くようにしておく必要があります。

オープンソースで安心

GitHubでソースコードを公開。どのように動いているかを誰でも確認できます。

スクリーンショット

使い方

  1. 1. アプリをインストールします。（Google Play または GitHub のリリースから）
  2. 2. VポイントPayにログインし、残高が表示されることを確認します。
  3. 3. ホーム画面を長押しし、「ウィジェット」を選択します。
  4. 4. 「VポイントPay残高ウィジェット」を選び、ホーム画面に配置します。
  5. 5. 配置後は、自動的に最新の残高が表示されます。
